The RGB Root Matriz Color Plotter

by Danielle Gauthier

This program is inspired by the ecofeminist pluriverse and feminist perspectives on family-making and crafting. It is used for telepathic communication and other forms of poetic linguistic navigation programming. The directional words are generated using a Python algorithm that analyzes color relationships and semantic meanings to suggest linguistically resonant navigation paths.

To use this deterministic model for sequence optimization:

  1. Enter an obstacle in the first textbox (optional).
  2. Select the type of pathway you want to explore (optional).
  3. For each color, enter how your obstacle relates to the directional words (optional), or leave as is to use the default directions.
  4. Hit Expose and imagine the possibilities.
